  1. Photo Album

    This is just a Photo Album which is made using HTML, CSS and bootstrap. This is also responsive to all type of devices. This is a simple project which displays the photos of the user in the form of album. When the user right clicks on the photo, a menu will be displayed which contains the options like increase brightness, duplicate the image, make the image reddish and so on. The user can select the option and the image will be changed according to the option selected by the user.

  2. Tic Tac Toe

    This is just a Tic Tac Toe game which is made using HTML, CSS, JavaScript and bootstrap. This is also responsive to all type of devices. This game is played by two players. The player who succeeds in placing three of their marks in a diagonal, horizontal, or vertical row is the winner. This game can also play by one player. In this case the player will play with the computer. The computer will play randomly.
